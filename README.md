# HamarTech Ren'Py Workshop Assets

This repository contains visual and audio assets created for the HamarTech Ren'Py workshop.

The assets are used in a tutorial video where participants learn the basics of Ren'Py and visual novel development.

Contents:
- Character art (GIGA)
- Backgrounds
- Sound effects and music

Assets are free to use for educational purposes.
